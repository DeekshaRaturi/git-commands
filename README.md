# Git commands


## git config
```sh
$git config --global user.name "deeksha raturi"
$git config --global user.email "diksharaturi56@gmail.com"
```
## git init
```sh
$git init
```
## git check version
```sh
$git --version
```
## git status
```sh
$git status
```
## git add file 
```sh
$git add "file name"
```
## git add all
```sh
$git add .
```
## git add remote repository
```sh
$git remote add origin "repository"
```
## git commit
```sh
$git commit -m "message"
```
## git push code
```sh
git push -u origin master
```
## git update code
```sh
$git fetch --all
```
## git checkout branch
```sh
$git checkout "branch name"
```
## git create branch
```sh
$git checkout -b "new branch name"
```
## git branch
```sh
$git branch
```
## git all branches including remote
```sh
$git branch -a
```
## git pull
```sh
$git pull "branch name"
```
## git clone
```sh
$git clone path/repository
```
## git clone from specific branch
```sh
$git clone -b "branch name" "repository name"
```
## git check remote 
```sh
$git remote
```
